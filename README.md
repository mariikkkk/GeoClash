# GeoClash

## 1. Описание проекта
GeoClash — это прототип игры, созданный на движке **Godot Engine 4**.  
Игрок управляет геометрической фигурой, сражающейся с волнами врагов-фигур.  
Цель — выжить как можно дольше, используя систему прокачки способностей.

## 3. Основные классы проекта
### Player
Этот класс отвечает за управление игроком, камерой и её эффектами, получением урона и взаимодействия с врагами.
В самом скрипте игрока можно поменять скорость движения.
Чтобы поменять начальное здоровье, нужно в сцене выбрать узел HealthComponent и в инспекторе изменить max_health

### HealthComponent
Управляет здоровьем всех объектов.
Можно изменять: max_health - здоровье объекта, take_damage() - логика получения урона, on_level_up() - прирост здоровья после повышения уровня (Доступно только для игрока)

### ExperienceManager
Отвечает за накопление и распределение опыта.
Можно изменять: current_exp - текущее количество опыта, target_exp - цель для достижения нового уровня, targer_after_lvlup - показывает на сколько увеличить цель для получения нового уровня, current_lvl - текущий уровень

### EnemySpawner
Отвечает за периодическое появление врагов на карте.
Вызывается по таймеру, может выбрать тип врага в зависимости от веса, не дает спавнится врагам за картой.
Можно изменять: расстояние появления, задержку и индикатор, логику появления и веса врагов, проверку коллизий.

### TriangleDash
Класс врага-треугольника. Преследует игрока и совершает быстрый рывок в его направлении.
Изменяемые параметры:
    max_speed — обычная скорость движения.
	dash_speed — скорость рывка.
	dash_trigger_dist — дистанция, с которой враг начинает подготовку.
	windup — длительность замаха перед рывком.
	dash_time — длительность самого рывка.
	cooldown — пауза после рывка.
	rotate_speed — скорость поворота к игроку.
